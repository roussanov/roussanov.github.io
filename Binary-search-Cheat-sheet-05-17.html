Binary search. Cheat sheet.

Alexey Roussanov

Forgot how to binary search? It's time to refresh!

Original statement: given an array of integers A sorted in ascending order, find the starting and ending position of a given x value.

Reframed statement: given that array, find the position starting from which the predicate evaluates to True.

i         -1 0 1 2 3 4  5 n
array      X 5 7 7 8 8 10 X
pred >= 8  0 0 0 0 1 1  1 1
pred >  8  0 0 0 0 0 0  1 1

Things to remember:

  1. There are 2 fake elements: before the first element and after the last one.
  2. There are 2 kinds of predicates: >= and >.
  3. The algorithms iteratively narrows the window [a, b] , starting with a = -1 and b = n.
  4. For >= predicate check the right bound b. For > check the left bound a.

Code (>= case):

a = -1 
b = n
while (a + 1 < b) {
  m = (a+b)/2
  if (A[m] >= x) {
    b = m
  }
  else {
    a = m
  }
}
if (b < n and A[b] == x) {
  predicate_true_since = b
}

Complexity: O(log n).

Warning: beware of integer overflow at m = (a+b)/2.
